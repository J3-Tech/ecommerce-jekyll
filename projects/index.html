liteArm / uArm - Open Robotic Arm

"Many hands make 'lite' work" goes the saying. That is the motto of the liteArm project, to create a common format robotic arm that can be used as a platform for the community to build off of. It was designed to be 3D Printable, and to run off of an arduino uno (plus shields).
